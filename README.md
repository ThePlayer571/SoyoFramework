## SoyoFramework

这是由ThePlayer571开发的渐进式Unity游戏开发框架。

## 目录

- [框架核心 Framework](Assets/SoyoFramework/Framework/README.md)
    - 这是框架的层级结构部分，提供了MVC设计模式的实现，也提供了一些基础的工具类
- [工具包 ToolKits](Assets/SoyoFramework/ToolKits)
    - 这是工具包，提供了一些常用的功能实现
- [UI面板管理工具 UIKit](Assets/SoyoFramework/OptionalKits/UIKit/README.md)
    - 这是一个UI面板管理工具，提出按照UIPage+UIView的形式管理跨场景、可复用的UI面板
- [UGUI拓展 SoyoUGUIKit](Assets/SoyoFramework/OptionalKits/SoyoUGUIKit)
    - 这是一些UGUI的拓展组件
- [游戏流程管理工具 ProcedureKit](Assets/SoyoFramework/OptionalKits/ProcedureKit)
    - 这是一个游戏流程管理工具
